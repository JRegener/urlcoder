# urlcoder

This is simple program to encode or decode url from UTF-8. 
The main perpose to make possible encoding/decoding URL from CMD.

## Example

### Encode
```
$ urlcoder.exe encode "Hello Günter"
Hello%20G%C3%BCnter
```

### Decode
```
$ urlcoder.exe decode "Hello%20G%C3%BCnter"
Hello Günter
```
